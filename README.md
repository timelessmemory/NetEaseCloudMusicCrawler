# NetEaseCloudMusicCrawler<br/>
## 网易云音乐爬虫 <br/>
* httpClient + Jsoup<br/>
* 单线程, 队列， log4j<br/>
* Top music<br/>
* 评论大于某值的smusic<br/>
* 所有爬取到歌曲的信息及评论<br/>
* 导出结果到excel<br/>
